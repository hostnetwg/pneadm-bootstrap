   CRITICAL INFORMATION FOR AI ASSISTANTS
═══════════════════════════════════════════════════════════════

⚠️  ALWAYS use 'sail' prefix for commands
    Example: sail artisan migrate (NOT: php artisan migrate)

🐳  Project runs in Docker via Laravel Sail on Windows WSL2

🔌  Database port: 3307 (host), 3306 (container)
    Always use 'mysql' as DB_HOST in .env

🎨  Frontend: Bootstrap 5.3.3 (not Tailwind)

⚡  Laravel version: 11.31+ (use modern syntax)

📦  Build tool: Vite 6.0 (not Mix)

🧪  PHP version: 8.4 (in container)
